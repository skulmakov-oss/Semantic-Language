# E0244: selected import symbol not found

## What it means

A symbol listed in `Import "..."{...}` does not exist in dependency exports.

Typical diagnostic includes wording like:
`selected import symbol 'X' not found`.

## Example

```exo
Import "dep.exo" { MissingSymbol }
```

## How to fix

1. Fix the symbol name typo.
2. Export the symbol in dependency module.
3. Replace select import with an actually exported symbol.

## Notes

Validation is performed against dependency `ExportSet`.
