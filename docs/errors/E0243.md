# E0243: symbol re-export cycle

## What it means

A re-export chain forms a cycle at symbol level.

Typical diagnostic includes wording like:
`symbol re-export cycle detected`.

Current direct-cycle wording includes a deterministic chain trace such as:
`a.sm -> b.sm -> a.sm`.

## Example

```exo
// a.sm
Import pub "b.sm"
// b.sm
Import pub "a.sm"
```

## How to fix

1. Break the cycle by exporting a local declaration directly.
2. Remove one re-export edge in the cycle.
3. Keep dependency direction acyclic for exported surface.

## Notes

The diagnostic includes a deterministic chain trace to show where the cycle
closes. The trace follows the current re-export recursion order.
