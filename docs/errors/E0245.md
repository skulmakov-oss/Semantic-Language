# E0245: invalid selected import form

## What it means

Import select list violates aliasing, form, or selected-symbol kind rules.

Typical diagnostics include:
`duplicate selected import alias`, wildcard/select conflict, or
`selected import symbol 'X' kind mismatch: expected ..., found ...`.

## Example

```exo
Import "dep.sm" { A as X, B as X }
```

or

```exo
Import "dep.sm" * { A }
```

## How to fix

1. Use unique aliases inside one select list.
2. Do not combine wildcard `*` with explicit select list.
3. Match the selected symbol kind to the requested kind qualifier such as
   `Entity:Foo` or `Law:Bar`.
4. Split invalid import into two valid imports if needed.

## Notes

This error is policy-level validation of one import statement.
