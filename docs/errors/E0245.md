# E0245: invalid select aliasing/import form

## What it means

Import select list violates aliasing or form rules.

Typical diagnostics include:
`duplicate selected import alias` or wildcard/select conflict.

## Example

```exo
Import "dep.sm" { A as X, B as X }
```

or

```exo
Import "dep.sm" * { A }
```

## How to fix

1. Use unique aliases inside one select list.
2. Do not combine wildcard `*` with explicit select list.
3. Split invalid import into two valid imports if needed.

## Notes

This error is policy-level validation of one import statement.
